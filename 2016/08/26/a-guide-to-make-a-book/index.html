『极简教程』gitbook写书，同步github

发表于 2016-08-26   |   分类于 互联网   |  

2016年8月11日 23:57:31
今天终于把《搞定人》整理完了，不说融会贯通吧，把笔记都分类整理了。
看知笔墨上书的呈现挺好的，章节明显，排版优美。也很适合《搞定人》笔记，赶紧整整看，可以和小伙伴分享搞定人的笔记。

用 minimal action knowledge 建立一个 minimal braggable product （最小可炫耀作品）

【工具】

  • github帐号
  • gitbook帐号
  • gitbook客户端（windows示范）

【MAK】最少必要知识
5分钟内学会使用markdown编辑器

【原理】
Gitbook 是基于 Node.js 的命令行工具，用来创建漂亮的电子书，它使用 Markdown 语法来撰写内容，用 Git 进行版本控制，且可以托管在 Github 上。Gitbook 可以将作品编译成网站、 PDF、 ePub 和 MOBI 等多重格式。

【步骤】
1、下载个客户端，登录，新建一本书，书写即可。

2、gitbook客户端上写好的书怎么导入github中？如图所示。github新建repo，把git地址复制到gitbook。

3、更新到github的书传到gitbook展示。登录gitbook刚才创建的书的地址，设置github repository，设置webhook；

登录github，设置webhook。

过程中可能会出现的问题，相信你都会解决的。
× GitHub webhook has not been added to the repository
√ Integration with this GitHub repository is working

【效果】
Atom+Github客户端=GitBook Editor编辑器，github更新gitbook。经过以上步骤后，就可以在本地修改文件，直接add，save，publish & sync。修改新书的同时，github会同步更新，gitbook.com上对应的书籍也会同步更新。
